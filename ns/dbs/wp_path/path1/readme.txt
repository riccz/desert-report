This path follow a sawtooth trajectory, in a square 100x100 m^2. 
For an ROV that move at speed of 1 m/s, each waypoint is sent when the previous one is supposed to be achieved, plus a guard time tg. The value of tg is reported in each file name (e.g. path_tg2s.csv has a tg = 2 s).
